WriteCraft : Ai content creator

Project Info:

WriteCraft is an AI-based content creation website.

Feature
  • Dashboard
  • Registration & login with email
  • Login with google
  • Create fully custom text editor for client
  • Download Text to pdf
  • Extract docx file content to text editor
Technology
  • React.js (Hook, JSX, Context-Api, Route and others)
  • ChatGPT
  • Nodejs
  • Firebase (Authentication)
  • Tawilwind
  • Html
  • Css3
